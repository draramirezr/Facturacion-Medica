# Configurar Repositorio Remoto

## Si tu repositorio está en GitHub:

```bash
git remote add origin https://github.com/TU_USUARIO/Facturacion-Medica.git
git branch -M main
git push -u origin main
```

## Si tu repositorio está en GitLab:

```bash
git remote add origin https://gitlab.com/TU_USUARIO/Facturacion-Medica.git
git branch -M main
git push -u origin main
```

## Si ya tienes el repositorio configurado:

Solo necesitas hacer push:
```bash
git push origin master
```

## Verificar remoto actual:

```bash
git remote -v
```
